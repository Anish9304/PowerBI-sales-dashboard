# Project Title
Sales Dashboard using Power BI

## Project Overview
This project analyzes sales performance, profit trends, category insights, and regional performance using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Category Analysis
- Region-wise Performance
- Interactive Filters

## Key Insights
- Technology category generated highest profit
- Consumer segment contributed maximum sales
- Q4 recorded highest sales growth
