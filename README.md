# 📊 Sales Dashboard using Power BI

## 📌 Project Overview
This project is an interactive **Sales Dashboard** developed using **Power BI** to analyze sales performance, profit trends, customer segments, regional growth, and category-wise business insights.

The dashboard helps transform raw sales data into meaningful visual insights for better business decision-making. It provides a detailed understanding of yearly sales trends, state-wise performance, segment contribution, and category profitability.

---

# 🎯 Objective
The main objective of this project is to:

- Analyze overall business sales performance
- Identify high-performing regions and categories
- Track yearly and quarterly sales trends
- Understand customer segment contribution
- Discover profit-generating categories and sub-categories
- Build an interactive and user-friendly dashboard for business analysis

# 📈 Dashboard Pages & Features

---

## 1️⃣ Overall Sales Dashboard

### Features
- Total Revenue KPI
- Total Profit KPI
- Total Quantity Sold KPI
- Sales and Profit by Year
- Quarterly Sales Analysis
- State-wise Sales Map
- Segment-wise Sales Contribution
- Category-wise Profit Analysis
- Date Slicer for Dynamic Filtering

### Key Insights
- Total Revenue reached approximately **$2.30M**
- Total Profit exceeded **286K**
- Sales showed continuous growth from **2011 to 2014**
- Q4 generated the highest sales across all years
- Consumer segment contributed the highest sales

---

## 2️⃣ Region and State Level Analysis

### Features
- Region Filter
- State Filter
- Year Filter
- City-wise Profit Table
- State-wise Category Sales Comparison
- Profit Trend by Category
- Segment-wise Sales Comparison
- Category & Sub-category Performance

### Key Insights
- California and New York are top-performing states
- Technology category generated maximum profit
- Some cities recorded negative profit despite strong sales
- Consumer segment dominated across regions

---

## 3️⃣ Category and Sub-Category Analysis

### Features
- Category & Sub-Category Filters
- Sales & Profit Treemap
- Segment-wise Quantity Distribution
- Sales vs Quantity Scatter Chart
- Quarterly Profit Trend
- Quarterly Sales Trend

### Key Insights
- Technology category delivered the highest profitability
- Furniture category showed lower profit margins
- Q4 consistently performed best in both sales and profit
- Consumer segment had maximum quantity contribution
